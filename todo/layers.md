# Pattern Hirachie / Pattern Layers (TODO)

- New section in the toolbar

Basically it's a way to build a hirachy in patterns, meaning you can place a pattern on top of another pattern and then the pattern on top get's seamed exactly to the pattern below (to create for example little bags in shirts or similar, or bags in trousers, etc.)

It should be the oposite to the seam tool, because everything is automatically seamed to the pattern below in the hirachy, and you are able to OPT OUT of seaming to create a bag that is only open on top for example.

- Each Pattern should be automatically in a hirachy, but default to 1 deep and you can add other pattern to that hirachy somehow (probably a menu, or colors, or numbers etc.)

- In a hirachy, each pattern has a number (1 is the bottom), every higher number get's seamed to the number below it, there can be multiple patterns per hirachy number (for example two bags)

- Somehow this should be clear to the user and easy to operate from a UX and UI perspective
